XTree diff + alogrithm.

## data structure

| property | description |
| --- | --- |
| label | 节点标签 |
| type | 节点类型 |
| value | 节点的值 |
| text | 文本节点的内容 |
| nMD | 节点的消息摘要 |
| tMD | 子树的消息摘要 |
| iMD | 节点的 ID 消息摘要 |
| nPtr | 匹配的节点指针 |
| Op | 编辑操作 |
